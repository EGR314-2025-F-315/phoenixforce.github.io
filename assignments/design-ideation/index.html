Goal of the Project

Develop a modular wildfire response system to improve early detection and monitoring of hot spots through distributed mobile sensor modules linked to a base display module. Using ESP‑Mesh networking, multiple units relay temperature, smoke, CO, and related indicators to a central, glove‑friendly interface for real‑time situational awareness and faster mitigation.

Success metrics
  • ≤ 1 s alert latency (lab target)
  • ≥ 99% message delivery over ≥ 100 trials
  • ≥ 95% detection accuracy for chosen condition
  • ≤ 3 actions to complete the core task

Audience of the Project

Primary: frontline wildfire responders (firefighters, fire‑watch, emergency management) who need rugged, legible, internet‑optional tools. Secondary: researchers and community stakeholders for monitoring and education.

Product Description

Mobile sensor units with GPS and a raiseable sensor mast form a distributed detection fabric. An ESP‑Mesh backbone relays readings to a rugged base module with multi‑modal alerts and glove‑friendly controls. Roles align to the course structure: networking, HMI, sensing/actuation, and systems integration (power, runtime, terrain navigation).
