Summary

Executive Summary

Mission

TerraGuard rover + screen HMI that monitors air quality, gas signatures, and location, while streaming video over a rover-hosted Wi‑Fi AP.

  • Two-PCB system: rover (sensing/compute) + screen (HMI).
  • All data flows via rover access point using HTTP.
  • LEDs/speaker provide clear status for safety conditions.

Highlights

  • Reliable AP link with continuous metrics + MJPEG stream.
  • Feedback loops: gas readings drive LEDs and alerts.
  • Power rails validated on final rover PCB.
  • Team roles span RF, sensing, power, drive, and HMI.
Architecture

System Architecture

Communication

  • ESP32-WROOM-32U hosts AP (TerraGuard-AP, 192.168.4.1).
  • Screen ESP32-S3 joins as STA; polls via HTTP GET.
  • Endpoints: /metrics (JSON telemetry) + /stream (MJPEG).
  • Two-way flow: rover serves, screen drives periodic pulls.

Hardware Stack

  • Rover PCB: ESP32-WROOM-32U, BME688, MQ-2, GPS (NEO-6M), camera.
  • Actuators: WS2812B LEDs, motor driver, status indicators.
  • Screen PCB: 7" ESP32-S3 HMI, LED strip, speaker alerts.
  • Power: battery → charger → 5V → 3.3V regulators (rover).
Testing

Verification & Results

Demonstrations

  • Wireless comms over rover AP with live metrics + video.
  • LEDs on rover and screen react to gas classifications.
  • Speaker alarms for dangerous gas or AP loss.
  • Motor driver operation verified on final PCB.
  • Power rails measured at 5V and 3.3V with multimeter.

Observations

  • Stable AP uptime with continuous pull pattern from screen.
  • Gas classification maps to red/yellow/green/pink states.
  • GPS telemetry integrated for location awareness.
  • Thermal/power within expected ranges under load.
Outlook

Risks & Next Steps

Risks

  • Environmental variability affecting gas sensor accuracy.
  • AP interference or range limitations in crowded RF spaces.
  • Thermals on regulators/driver under sustained load.

Future Improvements

  • Adaptive gas calibration routines for changing conditions.
  • Optional command channel from screen → rover for controls.
  • Richer UI overlays for GPS traces and alert history.
  • Hardening enclosure for outdoor deployment.
